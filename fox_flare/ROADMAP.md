# Fox Flare — Feature Roadmap

## Search & Navigation
1. **Search bar** — fuzzy file/folder search with real-time results (Ctrl+F or Ctrl+L)
2. **Breadcrumb navigation** — clickable path segments in the nav bar instead of plain text
3. **Bookmarks / Favorites** — pin frequently-used folders to the sidebar
4. **Recent locations** — quick-access list of recently visited directories
5. **Jump to path** — autocomplete + suggestions as you type in the path bar

## File Operations
6. **Drag and drop** — move/copy files by dragging between panels or from external apps
7. **Multi-select** — Shift+click range select, Ctrl+click toggle select
8. **Batch rename** — regex or pattern-based renaming of multiple files
9. **File properties dialog** — permissions, size, modified date, MIME type
10. **Create new file** — quick new file with extension picker
11. **Undo/Redo** — reversible file operations (move, rename, delete)
12. **Progress indicator** — progress bar for copy/move/delete operations
13. **Duplicate file/folder** — one-click duplicate

## Preview & Content
14. **Preview pane** — side panel showing image thumbnails, text file previews, PDF first page
15. **Image thumbnails in grid view** — actual image previews instead of generic icons
16. **Quick Look** — spacebar to preview a file without opening it (like macOS)
17. **Syntax-highlighted text preview** — for code files
18. **Media info** — show image dimensions, video duration, audio bitrate

## UI & Customization
19. **Split pane / dual panel** — side-by-side directory views (Commander-style)
20. **Custom themes** — let users create or import color themes
21. **Column view** — Finder-style column navigation
22. **Sort options UI** — sort by name, size, date, type with visible toggle
23. **Status bar** — total items, selected count, disk usage at the bottom
24. **Keyboard shortcuts overlay** — `?` to show all hotkeys
25. **Right-click "Open With" picker** — actually functional app chooser
26. **Zoom slider** — smooth icon size scaling instead of fixed presets
27. **Hidden files toggle** — Ctrl+H to show/hide dotfiles

## Power Features
28. **Terminal integration** — embedded terminal panel at the bottom
29. **Batch file operations queue** — queue up multiple operations
30. **Archive support** — browse inside .zip, .tar.gz, .7z without extracting
31. **Network locations** — SFTP, SMB, FTP mount browsing
32. **Cloud storage integration** — Google Drive, Dropbox, etc. via API
33. **Git status indicators** — show modified/untracked/staged status on files
34. **Checksum calculator** — MD5/SHA256 from right-click menu
35. **Disk usage visualization** — treemap or sunburst view of folder sizes
36. **Watch folder** — auto-refresh when file system changes (inotify)

## System & Polish
37. **Desktop file associations** — register as default file manager via .desktop
38. **Trash management** — view and restore trashed files
39. **System tray integration** — background quick-access
40. **CLI companion** — `fox-flare /path` to open at a specific location
41. **Remembering window state** — restore size, position, last open tabs on relaunch
42. **Localization / i18n** — multi-language support
