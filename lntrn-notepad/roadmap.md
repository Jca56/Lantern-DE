# lntrn-notepad WYSIWYG Roadmap

Transforming lntrn-notepad from "notepad with formatting" into a proper MS Word-style document editor.

## Phase 1: Core Document Formatting
- [ ] **Paragraph Alignment** — Left / Center / Right / Justify per paragraph
- [ ] **Word-Boundary Wrapping** — Wrap at spaces/hyphens, not mid-word
- [ ] **Line Spacing** — 1.0x, 1.15x, 1.5x, 2.0x per paragraph
- [ ] **Paragraph Spacing** — Configurable before/after gap
- [ ] **Indentation** — First-line indent, left/right margins per paragraph

## Phase 2: Styles & Structure
- [ ] **Heading Styles** — H1, H2, H3, Body, Quote presets (paragraph + character attrs)
- [ ] **Bullet Lists** — Auto-indent with bullet prefix rendering
- [ ] **Numbered Lists** — Auto-numbering with indent, renumber on reorder
- [ ] **Tab Stops** — Real tab characters at configurable stops, indent/outdent for lists

## Phase 3: Toolbar & UI
- [ ] **Alignment Buttons** — Left/Center/Right/Justify toggle group in toolbar
- [ ] **Heading/Style Dropdown** — Quick style selector
- [ ] **Text Color Picker** — UI for the existing color attr
- [ ] **Highlight Color Picker** — UI for the existing bg_color attr
- [ ] **List Toggle Buttons** — Bullet/Numbered in toolbar
- [ ] **Line Spacing Control** — Dropdown or button in toolbar
- [ ] **Right-Click Context Menu** — Cut/Copy/Paste/Select All + formatting

## Phase 4: Page Layout
- [ ] **Page View** — White page rect on gray background with visible margins
- [ ] **Page Margins** — Configurable top/bottom/left/right
- [ ] **Page Break Indicators** — Visual separator between pages
- [ ] **Zoom Controls** — Independent of DPI, toolbar or Ctrl+scroll

## Phase 5: Import & Export
- [ ] **DOCX Import** — Read .docx with formatting preserved
- [ ] **PDF Export** — Render document to PDF
- [ ] **Preserve Formatting on Save** — Custom format or RTF for lossless save/load

## Phase 6: Stretch Goals
- [ ] **Inline Images** — Insert and render images in document flow
- [ ] **Ruler** — Top ruler showing margins, tab stops, indents
- [ ] **Font Family Picker** — Multiple font families per document
- [ ] **Tables** — Basic table support
- [ ] **Spell Check** — Wavy underline on misspelled words
